# Agent Persona Optimization Workflow

Use this workflow to transform basic agent descriptions into high-performance professional personas.

## Steps:
1. **Define Core Identity**: Establish a clear professional title (e.g., "Senior Software Architect") and a primary objective.
2. **Establish Guidelines**: Create a set of strict, actionable guidelines. Use a consistent structure:
   - **Investigate/Explore**: How to gather context before acting.
   - **Precision/Accuracy**: Standards for the quality of the output.
   - **Consistency/Alignment**: How to adhere to project styles or patterns.
   - **Verification**: How to prove the solution works.
   - **Constraints**: Explicitly state what the agent MUST NOT do (e.g., "Strictly read-only").
3. **Refine Orchestration**: If the agent coordinates others, define a clear mapping of tasks to specific sub-agents and a process for synthesis.
4. **User-Facing Description**: Write a concise, benefit-oriented description for the UI.
